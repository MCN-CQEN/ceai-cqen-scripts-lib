# Scripts communs de développement
Ce dépôt sert de référentiel centralisé pour tous les scripts d'automatisation, utilitaires et outils de gestion courants utilisés au sein du CEAI (Centre d'Expertise Appliquée en Innovation du CQEN).

L'objectif est de fournir un point d'accès unique, versionné et documenté pour aider les équipes à rationaliser leurs processus et à maintenir la cohérence dans l'automatisation.

## Contenu

- [actions](./actions/README.md): Scripts communs et génériques pour GitHub Actions.
- [exemples](./exemples): Scripts qui montrent comme faire l'appel aux scripts communs.
